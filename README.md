## 识别精不精准？三大引擎说了算

iText 支持腾讯、百度、Google 三大 OCR 引擎，用户可以根据需要手动切换。

官方和大量用户的实测反馈一致：**对普通自然语言文本（书籍段落、新闻稿、正式文件），识别准确率非常高，正常场景下接近满分水平**。腾讯引擎对中文表现尤为稳定，Google 引擎则在日文、其他小语种等场景更有优势。

不过也要说实话：如果你想识别程序代码、特殊符号或者竖版排版，识别结果就没那么稳定，偶尔需要手动校对。这其实是当前 OCR 技术的普遍局限，不是 iText 独有的问题。

软件对此也提供了**双栏对比校对**功能——识别结果和原图并排显示，方便核对修改，比其他工具单独弹出文字框要贴心一些。
